Bug Report: Login Button Unresponsive

Bug ID: BR-001

Severity: High

Priority: High

Summary

The Login button does not respond after entering valid credentials.

Steps to Reproduce

  1. Open the login page.
  2. Enter valid credentials.
  3. Click the Login button.

Expected Result

User should be logged in successfully.

Actual Result

Nothing happens when the Login button is clicked.
